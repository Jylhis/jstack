`/guard` activates full safety mode — combining [`/careful`](/skills/careful) (destructive command warnings) and [`/freeze`](/skills/freeze) (directory-scoped edits) in one command.

## When to use it

- Working on production systems
- Debugging live issues
- Any time you want maximum safety

## Example

```
/guard src/payments

Claude: Safety mode active:
        - Destructive command warnings enabled (/careful)
        - Edits restricted to src/payments/ (/freeze)
```
