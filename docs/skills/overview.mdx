Every jstack skill is a specialist role. They follow the sprint order: Think &rarr; Plan &rarr; Build &rarr; Review &rarr; Test &rarr; Ship &rarr; Reflect.

## Specialist skills

| Skill | Your specialist | What they do |
|-------|----------------|--------------|
| `/office-hours` | **YC Office Hours** | Six forcing questions that reframe your product before you write code. |
| `/plan-ceo-review` | **CEO / Founder** | Find the 10-star product hiding inside the request. |
| `/plan-eng-review` | **Eng Manager** | Lock architecture, data flow, diagrams, edge cases, and tests. |
| `/plan-design-review` | **Senior Designer** | Rate each design dimension 0-10, explain what a 10 looks like. |
| `/design-consultation` | **Design Partner** | Build a complete design system from scratch. |
| `/review` | **Staff Engineer** | Find the bugs that pass CI but blow up in production. |
| `/investigate` | **Debugger** | Systematic root-cause debugging. No fixes without investigation. |
| `/design-review` | **Designer Who Codes** | 80-item visual audit + fix loop. Atomic commits. |
| `/qa` | **QA Lead** | Open a real browser, find bugs, fix them, re-verify. |
| `/qa-only` | **QA Reporter** | Same methodology as /qa but report only. |
| `/cso` | **Chief Security Officer** | OWASP Top 10 + STRIDE threat model. |
| `/ship` | **Release Engineer** | Sync main, run tests, audit coverage, push, open PR. |
| `/land-and-deploy` | **Release Engineer** | Merge, deploy, verify production health. |
| `/canary` | **SRE** | Post-deploy monitoring loop. |
| `/benchmark` | **Performance Engineer** | Baseline load times, Core Web Vitals, and resource sizes. |
| `/document-release` | **Technical Writer** | Update all docs to match what you shipped. |
| `/retro` | **Eng Manager** | Team-aware weekly retro with per-person breakdowns. |
| `/browse` | **QA Engineer** | Real Chromium browser, real clicks, ~100ms per command. |
| `/setup-browser-cookies` | **Session Manager** | Import cookies from your real browser. |
| `/autoplan` | **Review Pipeline** | Runs CEO &rarr; design &rarr; eng review automatically. |

## Power tools

| Skill | What it does |
|-------|-------------|
| `/codex` | Independent code review from OpenAI Codex CLI. Cross-model analysis. |
| `/careful` | Warns before destructive commands. |
| `/freeze` | Restrict file edits to one directory. |
| `/guard` | `/careful` + `/freeze` combined. |
| `/unfreeze` | Remove the `/freeze` boundary. |
| `/setup-deploy` | One-time deploy configuration. |
| `/jstack-upgrade` | Self-update to latest version. |
