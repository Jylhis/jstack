After [installing jstack](/getting-started/installation), try these skills to see what jstack can do:

## 1. Start with Office Hours

```
/office-hours
```

Describe what you're building. The agent asks six forcing questions to reframe your product before you write code. It pushes back on your framing, challenges premises, and generates implementation alternatives.

## 2. Review a feature idea

```
/plan-ceo-review
```

CEO-level review of any feature idea. Finds the 10-star product hiding inside the request.

## 3. Review any branch with changes

```
/review
```

Staff-engineer-level code review. Finds bugs that pass CI but blow up in production. Auto-fixes the obvious ones.

## 4. Test your app

```
/qa https://staging.myapp.com
```

Opens a real Chromium browser, clicks through flows, finds bugs, fixes them with atomic commits, and re-verifies. Auto-generates regression tests for every fix.

## 5. Ship it

```
/ship
```

Syncs main, runs tests, audits coverage, pushes, and opens a PR. One command.

<Info>
Stop there. You'll know if this is for you after these five commands.
</Info>
